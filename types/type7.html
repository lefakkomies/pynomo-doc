Type 7¶

Type 7 is “angle” nomograph that has functional relationship:

\[\frac{1}{F_1(u_1)}+\frac{1}{F_2(u_2)}=\frac{1}{F_3(u_3)}\]

Simple example¶

This simple example plots nomograph for equation:

\[1/ u_1 + 1/u_2 = 1 / u_3\]
